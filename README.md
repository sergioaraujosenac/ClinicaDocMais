Clínica DocMais

Sistema moderno para gestão de clínicas médicas, focado em organização, eficiência e praticidade no atendimento.

✨ Funcionalidades

A Clínica DocMais foi desenvolvida para facilitar o dia a dia de clínicas e consultórios médicos. Confira o que o sistema oferece:

* 👤 Cadastro completo de pacientes
* 🩺 Cadastro e gerenciamento de médicos
* 📅 Agendamento de consultas intuitivo
* 📋 Controle de prontuários médicos
* 🕓 Histórico detalhado de atendimentos
* 🔐 Sistema de login e autenticação
* 📊 Painel administrativo
* 🚀 Instalação

Siga os passos abaixo para rodar o projeto na sua máquina:

# Clone o repositório
git clone https://github.com/seu-usuario/clinica-docmais.git

# Acesse a pasta
cd clinica-docmais

# Atualize o banco de dados
dotnet ef database update

# Execute o projeto
dotnet run

🔧 Importante:
Configure a conexão com o MySQL no arquivo appsettings.json antes de rodar.
